# Car Rental & Selling Platform Frontend

This is the frontend for a modern car rental and selling platform, built with React, TypeScript, and Vite.

## Features
- Browse, rent, and buy cars
- User authentication and profile management
- Car listing management (for sellers)
- Responsive and modern UI
- Fast development with Vite

## Project Structure
```
src/
  components/   # Reusable UI components
  pages/        # Page-level components (Home, CarDetails, etc.)
  routes/       # App routing configuration
  assets/       # Images, icons, and static files
  utils/        # Utility functions and helpers
```

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production
```bash
npm run build
```

### Linting & Formatting
- ESLint and Prettier are recommended for code quality and consistency.
